# Intastyl

A modern website for the Intastyl hair salon, built with Astro.js. This repository is a hands-on learning project dedicated to exploring the capabilities of Astro for building fast, content-focused websites.

## Commands

All commands are run from the root of the project, from a terminal:

| Command         | Action                                      |
| :-------------- | ------------------------------------------- |
| `npm install`   | Installs dependencies                       |
| `npm run dev`   | Starts local dev server at `localhost:4321` |
| `npm run build` | Build your production site to `./dist/`     |
